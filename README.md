# YouTube Spam Comment Detection 🚫

## Overview
Detects YouTube spam comments using Machine Learning and NLP.

## Tools Used
- Python, Scikit-learn (SVM)
- TF-IDF Vectorizer
- Streamlit (GUI)
- Seaborn, Matplotlib

## Results
- Accuracy: 95%+
- Dataset: 1956 comments (5 CSV files)

## How to Run
pip install streamlit scikit-learn imbalanced-learn
streamlit run GUI.py
